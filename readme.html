このライブラリはブラウザのJavaScriptで使うことを目的とした数学用ライブラリです
バンドルはWebpackによって行われており基本的に外部ライブラリは使っていません
ソースディレクトリはsrc、バンドルはdistに出力されます。

四則演算

四則演算はadd,sub,mul,divであり、それぞれ2個以上の任意の引数を受けられます
複素数Complexや線形代数クラスVectorやMatrixにも対応してあります
複素数と数字の演算は複素数で返されるようになってあります

計算用クラス

線形代数クラスは複素数も要素として持てるようになってあります
殆どのメソッドは複素数にも対応してあります
